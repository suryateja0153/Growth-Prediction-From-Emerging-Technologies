To machine translate a file in Text United click on „Create a project“ button and select „File Upload“ Choose the files from your computer, select the source language of your file and the target language you want to translate the file into. In the next step select the option „Machine“ and click on translate. Wait for a few moments and click on Download to retrieve your translated file. Done! Thank you for watching.
