 PiCar-S with 3 sensor modules SunFounder PiCar-S is a cool Raspberry Pi-based smart sensors robot car. Equipped with three sensor modules including ultrasonic obstacle avoidance, light follower, and line follower, you can understand the fundamental principles of robotics Line Following and master the programming with the visual programming language software Dragit (also Python code) we provide with this DIY project. It's designed for STEM education. Ultrasonic obstacle avoidance  Detecting range Light following   Easy to assemble Fun of DIY
