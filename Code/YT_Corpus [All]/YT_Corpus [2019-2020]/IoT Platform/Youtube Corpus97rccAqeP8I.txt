 In this video tutorial, we will show you how to create a template component. The template component is an advanced component that can contain more than one basic component and graphic shapes. First, go to template component Editor, create new template component "File - Edit Templates". Insert e.g. "Value Indicator", click "+" sign and close popup. Choose "Value Indicator" from the available templates list. First, we will add rectangle 100x25 px. Insert component name "Name BG". Set background, size and position. Create another one rectangle, 100x25 px. Set size and position. Insert component name "Value BG". Insert "Text Tool" component. Type "Value". Set font size, for component name insert "Value". Set text align, set component position to the center of the bottom rectangle, set text color. Insert another "Text Tool" component. Type "Name". Set font size, for the component name, insert "Name". Set text align, set component position to the center of the top rectangle, set text color. Set canvas size to 100x50px. Click the "Save" button. Go to the "Web Client". Go to the "Indas" components menu, from Template components choose "Value Indicator" and add it to your screen. In "Lista" click "Name". For "Text" insert "Tank Level [m]". In "Lista" click "Value" and click "Choose Variable". Choose "Random.Int1" add it and click "Save" button. Click the "Save" button, go to the "Client page" and reload it. "Value Indicator" show tank level value.
