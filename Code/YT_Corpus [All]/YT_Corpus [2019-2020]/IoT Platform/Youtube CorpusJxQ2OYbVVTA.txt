 In this video tutorial, we will show how to use drop-down selection. Changing drop-down value, we will change the graphic list image, slider value, I/O field, and text tool value. From "Indas" components menu choose drop down list, add a component to your screen set size and position. Click "Choose Variable" and assign variable, in our case that is "Bucket Brigade.Int1". Click "Manage States". Add 4 states. 0 - OFF, 1 - AUTO, 2 - ERROR, 3 - WARNING. Click the "Save" button, go to the "Client page" and reload it. When we change, drop-down selection values graphic list change image.
