 In this video tutorial, we will show you how to create and use Face-plate. To create face-plate go to face-plate Editor, create new face-plate "File - Edit Faceplates" For name insert "Tank Level", click "+" sign and close popup. From the face-plate list choose "Tank Level". Set canvas background color. Take "Text Tool" component and insert text: "Tank Level:" Take another "Text Tool" component and insert text: "VAL". Align components. Select "VAL", then click "Choose Variable". Assign "Random.Int1" variable, and click the "Save" button. From ""Indas" Component" menu insert template component "Image Button Mobile". In "Lista" select "Image". Click "Choose Image"and from list choose e.g. "Close" . Click the "Save" button. In "Lista" select "Button". Click "Manage Actions". For action, choose "Close popup" and click the "Save" button. Set canvas size too, e.g. 307x82, select text colour and click the "Save" button. Go to the "Client page". From ""Indas" Components" menu choose "Face-plate Opener". Position component over tank level vertical bar. Resize face-plate opener. Click "Choose Faceplate", from available faceplate list choose "Tank Level" and click the "Save" button. Set faceplate opacity to "0". Click the "Save" button, go to the "Client page" and reload it. Click on tank "Tank Level" vertical bar. The previously created faceplate is shown. Tank level value is shown, you can move faceplate. To close faceplate, click "X" on faceplate popup.
