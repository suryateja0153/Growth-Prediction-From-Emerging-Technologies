 In this demo, you'll learn how to customize the color scheme and font size in thinkorswim. After logging in, select Setup in the upper-right corner of the screen. Select Application Settings to open the User Setup window. Select Look and feel to bring up the appearance choices, which include Color scheme and Font size. Select Color scheme to see available choices: Dark, Light, Old School TOS, Chinese Dark, and Chinese Light. To create your own, select an existing color scheme as a starting place, and update the color values. Name your color scheme, and select Save. To adjust the font size, select Font size. Available font sizes are Small, Medium, Large, Larger, Very Large, and Extra Large. After you finish customizing the appearance of the platform, select Apply settings at the lower right of the User Setup window.
