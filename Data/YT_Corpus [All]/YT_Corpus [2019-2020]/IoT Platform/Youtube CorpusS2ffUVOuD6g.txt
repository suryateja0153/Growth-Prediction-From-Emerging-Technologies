 This screen contains a list of all configurations that the user saved. The checkbox "Show mobile configurations" preview all mobile configurations that the user saved. At the bottom right corner, we have pager which can navigate through configurations list. User has the option to set one of previously saved configuration to current. Select the desired configuration and click "Set selected as current". To delete one of the configurations select the desired configuration and click "Delete selected" button. In popup "Delete configuration" select "Yes" to delete the configuration. The user also can delete all configuration except last 20. To execute this action click "Delete all except last 20". In popup "Delete configuration" select "Yes" to delete configurations.
